Schedule Builder
================

Schedule Builder will take a list of students and assign students to teachers
and to classes based on:

    * Core Classes
    * Chosen Electives
    * Grade Levels

Features
--------

    * Custom Input
    * G.U.I
    * Searching for People and Classes
    * View Classes and its Students / Teachers
